# Muhammad Alfarizi Habibullah — Portfolio

Personal portfolio website showcasing projects, skills, and contact information. Built with modern web technologies and a focus on clean, responsive UI.

## Overview

This project presents professional experience as a Full-stack Developer and Android Developer. It includes:
- Homepage with personal profile and navigation
- Expertise section with Devicon monochrome icons
- Portfolio gallery for selected projects
- Contact section with a functional form and direct email link

## Tech Stack

- React + Vite
- TypeScript
- Tailwind CSS
- shadcn/ui (Radix UI components)
- Devicon (technology icons)

## Getting Started

### Prerequisites
- Node.js 18+ (LTS recommended)
- npm 9+

### Installation
```bash
git clone https://github.com/V60Code/portfolio.git
cd portfolio
npm install
```

### Development
```bash
npm run dev
```
Runs the development server with hot reload.

### Build
```bash
npm run build
```
Generates a production build in `dist/`.

### Preview
```bash
npm run preview
```
Serves the built application locally for verification.
